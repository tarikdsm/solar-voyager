# T0055 Navball Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use
> superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Deliver an allocation-free, snapshot-backed bottom-center navball whose
orbital markers are verified against a known LEO frame.

**Architecture:** A pure `src/ui/navballProjection.ts` writer converts the
existing float64 snapshot into a preallocated projection buffer. The sampled HUD
signal graph copies its scalar slots into leaf signals, and a static SVG component
updates only transforms and opacity.

**Tech stack:** TypeScript strict, Vitest, Preact signals, SVG/CSS, Playwright
Chromium, Vite.
