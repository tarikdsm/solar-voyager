# Solar Voyager

A realistic solar-system exploration game that runs in the browser.

Launch from the Alcântara Launch Center in Brazil, fight your way through the atmosphere under real physics, reach orbit, and travel anywhere in the solar system — every planet, moon, dwarf planet, asteroid and comet is there, at real scale, on real orbits, all pulling on your ship at once. Your ship has unlimited propulsion energy, but every m/s of delta-v you spend is metered on the HUD: orbital mechanics still rules.

**Play:** deployed on GitHub Pages at `https://tarikdsm.github.io/solar-voyager/` (after milestone M0).

## Highlights

- **Full n-body physics on the ship** — all ~50 bodies attract it simultaneously. Lagrange points, gravity assists and third-body perturbations are real, not scripted.
- **Real scale, real time** — with time warp from 1x to 10,000,000x.
- **2D launch phase** from Alcântara (2.37°S): manual throttle, pitch and gravity turn through a US Standard Atmosphere with Mach-dependent drag, then a seamless handoff to full 3D.
- **Blender-authored assets** — every model is generated by versioned, reproducible Python scripts.
- **Built by a team of AI agents** (Claude Code, ChatGPT, Grok) working from a shared task queue. See [docs/task-protocol.md](docs/task-protocol.md).

## Tech

Three.js + TypeScript + Vite, Preact HUD, Vitest, GitHub Actions → GitHub Pages. Physics is custom (float64, Dormand–Prince 5(4), analytic Keplerian rails baked from JPL Horizons).

## Development

```bash
npm install
npm run dev        # local dev server
npm test           # physics + unit tests
npm run build      # production build (base /solar-voyager/)
```

Asset generation requires Blender 4+ (`tools/blender/`), see [docs/asset-pipeline.md](docs/asset-pipeline.md).
