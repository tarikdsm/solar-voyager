# Sources — saturn

All maps: **Solar System Scope textures** (https://www.solarsystemscope.com/textures/) — **CC BY 4.0** (attribution required; based on NASA data). Downloaded 2026-07-15; originals in `assets/textures-src/saturn/`.

- `saturn_albedo.jpg` — 8k_saturn.jpg — CC BY 4.0 — unmodified copy
- `saturn_rings.png` — 8k_saturn_ring_alpha.png — CC BY 4.0 — unmodified copy; **8192×500 radial strip** (exceeds the guide's 2048×64 minimum), U = radial position across D→F rings, alpha = optical depth. Also the sampling source for the engine's close-range instanced ice-particle layer (task T0083).
- `saturn.glb` — generated by `tools/blender/build_saturn.py` (drafted live via blender-mcp, 2026-07-15). Nodes `saturn` (oblate 0.902, 160×80 hero mesh) + `saturn_rings` (annulus at real Cassini radii: D inner 1.110 R → F outer 2.327 R, 256×24 quads, radial-strip UVs, double-sided BLEND). 37,568 tris. Axial tilt (26.73°) applied by the engine.
