# Sources — earth

All maps: **Solar System Scope textures** (https://www.solarsystemscope.com/textures/) — **[CC BY 4.0](https://creativecommons.org/licenses/by/4.0/)** (attribution required; based on NASA elevation and imagery data). Downloaded 2026-07-15; originals kept in `assets/textures-src/earth/`.

- `earth_albedo.png` — 8k_earth_daymap.jpg — CC BY 4.0 — source download, checksum, metadata-free RGB conversion, and 8192×4096 normalization are reproducible through the `earth-albedo` recipe in `tools/fetch_textures.py`
- `earth_emissive_night.jpg` — 8k_earth_nightmap.jpg — CC BY 4.0 — unmodified copy
- `earth_clouds.jpg` — 8k_earth_clouds.jpg — CC BY 4.0 — unmodified copy (grayscale used as alpha of the cloud shell)
- `earth_normal.png` — 8k_earth_normal_map.tif — CC BY 4.0 — downscaled to 4096×2048, converted TIF→PNG
- `earth_detail_albedo.jpg` — procedural periodic micro-variation generated by `tools/generateDetailTextures.mjs` — project-authored — 1024×1024 seamless JPEG
- `earth_detail_normal.png` — procedural periodic height derivatives generated by `tools/generateDetailTextures.mjs` — project-authored — 1024×1024 seamless tangent-space PNG
- `earth.glb` — generated by catalog-driven `tools/blender/build_planet.py --id earth`; `build_earth.py` is its compatibility delegate. Nodes `earth` (surface) + `earth_clouds` (shell at 1.004 R); materials `mat_surface`, `mat_clouds`; no embedded images (ingest wires KTX2). 32,256 tris (budget ≤50k).

Credit line for the game credits screen: "Earth textures: Solar System Scope (solarsystemscope.com), CC BY 4.0."
