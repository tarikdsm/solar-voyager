# Sources — pluto

Maps: **NASA / JPL Photojournal PIA11707** (New Horizons Ralph/MVIC color global mosaic, July 2015 flyby) — **public domain** (U.S. government work). Credit: NASA / Johns Hopkins University Applied Physics Laboratory / Southwest Research Institute. Source page: https://www.jpl.nasa.gov/images/pia11707-pluto-color-map/

Original kept in `assets/textures-src/pluto/`.

- `pluto_albedo.jpg` — derived from the public-domain `PIA11707_pluto_color_map.jpg` (5926×2963 equirectangular color mosaic) by `tools/textures/preparePlutoMap.mjs`, seed 999 — resized to **2048×1024** (2k dwarf tier, exact 2:1) via Lanczos, then the black southern no-data mask and its JPEG acquisition edge were reconstructed with deterministic multi-octave 3D value noise sampled on the unit sphere and color-matched to the mapped boundary; a 5.5% two-dimensional feather removes the acquisition outline, while spherical sampling and longitude-converged polar rows prevent a UV seam or pinched pole; JPEG q92
- `pluto_detail_albedo.jpg` — procedural periodic ice/regolith micro-variation generated by `tools/generateDetailTextures.mjs`, seed 999 — Solar Voyager original asset; all rights reserved for project distribution — 1024×1024 seamless JPEG
- `pluto_detail_normal.png` — procedural periodic height derivatives generated by `tools/generateDetailTextures.mjs`, seed 999 — Solar Voyager original asset; all rights reserved for project distribution — 1024×1024 seamless tangent-space PNG
- `pluto.glb` — generated by `tools/blender/build_pluto.py` (also reviewable live via blender-mcp). Node `pluto` (UV sphere 64×32, radius 1.0 normalized); material `mat_surface` (roughness 0.78, non-metal ice/tholin surface); no embedded images (ingest wires KTX2). 3,968 tris (budget ≤15k). Axial tilt applied by the engine.
