# assets/ — Asset Source Workspace

This is where 3D models and source textures are authored by humans and tools.

- `models/<category>/<body-id>/` — authored `.glb` deliverables and external textures. The singleton `sun/` and `ship/` categories keep their files directly in the category directory. Read [models/MODELING-GUIDE.md](models/MODELING-GUIDE.md) before creating anything.
- `blender/` — committed `.blend` sources for hand-authored hero assets. Keep textures external, never packed.
- `textures-src/` — downloaded/processed source textures. Prefer a reproducible fetch script; commit source files only when no stable URL exists.

This directory is the source of truth. `public/assets/` is generated by
`npm run assets:ingest` (validation → Draco → KTX2 → budget check). Never edit
runtime output by hand or load the game directly from source assets.

Set `KTX_BIN` when KTX-Software 4.4.x is not on `PATH`.
